CHROME EXTENSION ICONS

You need to add three icon files to this directory:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can:
1. Create simple icons using any image editor (GIMP, Photoshop, Figma, etc.)
2. Use online icon generators like https://www.favicon-generator.org/
3. Use a simple colored square with "JT" text (for Job Tracker)

Temporary Solution:
For testing purposes, you can create simple colored PNG files or download placeholder icons.

Alternative:
If you don't want to create icons right now, you can temporarily remove the icon references from manifest.json.
